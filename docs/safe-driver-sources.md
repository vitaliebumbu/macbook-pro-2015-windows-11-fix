# Safe Driver Sources

## Use These First

Use official or well-known sources:

- Boot Camp Assistant in macOS
- Apple Software Update in Windows
- Brigadier, which downloads Boot Camp packages from Apple

Brigadier:

```text
https://github.com/timsutton/brigadier
```

## Community Tools Mentioned

Trackpad:

```text
https://github.com/imbushuo/mac-precision-touchpad
```

Fan control:

```text
https://github.com/bmats/fancontrol
```

Macs Fan Control:

```text
https://crystalidea.com/macs-fan-control
```

## Avoid These

Avoid:

- Random driver-pack websites
- Repacked Boot Camp installers
- Driver updater ads
- Download links from forum comments with no source
- Anything asking to disable antivirus

Bad driver packages can replace Apple-specific drivers with generic drivers and make the MacBook worse.
