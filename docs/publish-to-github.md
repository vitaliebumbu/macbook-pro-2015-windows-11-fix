## Upload With GitHub Web

1. Go to GitHub.
2. Click **New repository**.
3. Name it:

```text
macbook-pro-2015-windows-11-fix
```

4. Make it public.
5. Do not add a README on GitHub if you are uploading these files.
6. Upload the files from this folder.
7. Add the topics listed above.

## Publish With Git Later

If Git is installed:

```bat
git init
git add .
git commit -m "Add MacBook Pro 2015 Windows 11 fix guide"
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/macbook-pro-2015-windows-11-fix.git
git push -u origin main
```
